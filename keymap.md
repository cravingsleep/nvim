# Keymaps

A full list of all the keymaps (in normal mode unless otherwise stated):

### Vim

|Keys|Action|
|----|------|
|`<leader>w`|`:w`|
|`jj`|exit terminal mode|

### Tabs

|Keys|Action|
|----|------|
|`[b`|go to previous tab|
|`]b`|go to next tab|
|`<C-p>`|select tab using key|

### Git

|Keys|Action|
|----|------|
|`<leader>hd` |highlights the git diff|
